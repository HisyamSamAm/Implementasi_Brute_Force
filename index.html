Pencarian Pola dalam Teks

Brute force cocok untuk mencari pola sederhana dalam teks pendek atau ketika kita membutuhkan implementasi yang mudah dipahami.

Contoh Kode:
function cariPola(teks, pola) {
    for (let i = 0; i <= teks.length - pola.length; i++) {
        let cocok = true;
        for (let j = 0; j < pola.length; j++) {
            if (teks[i + j] !== pola[j]) {
                cocok = false;
                break;
            }
        }
        if (cocok) return i;
    }
    return -1;
}
